# 🏢 ربات کاریابی همراکار — بله

## ⚡ راه‌اندازی سریع (ویندوز)

### مرحله ۱: نصب Python
- دانلود و نصب **Python 3.10+** از [python.org](https://python.org)
- ✅ گزینه "Add Python to PATH" را بزنید

### مرحله ۲: دابل‌کلیک روی `run.bat`
- Notepad خودکار باز می‌شود
- دو مقدار زیر را پر کنید و ذخیره کنید:

```
BOT_TOKEN=توکن_بله_شما
ADMIN_IDS=آیدی_عددی_شما
```

### مرحله ۳: شروع ربات
- ربات خودکار شروع می‌کند
- پیام "ربات همراکار شروع شد..." را ببینید ✅

---

## 🔗 دریافت توکن و آیدی

### توکن بله:
1. در اپ بله جستجو کنید: `@BotFather`
2. پیام بفرستید: `/newbot`
3. توکن دریافت کنید

### آیدی عددی:
1. در اپ بله جستجو کنید: `@userinfobot`
2. هر پیامی بفرستید
3. آیدی عددی شما نمایش داده می‌شود
